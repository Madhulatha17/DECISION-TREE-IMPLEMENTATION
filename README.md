# DECISION-TREE-IMPLEMENTATION

## PROJECT OVERVIEW

This project implements a Decision Tree Classifier using Scikit-Learn on the Iris dataset. The model is trained to classify flower species based on sepal and petal measurements.

## FEATURES

- Data Loading
- Model Training
- Prediction
- Accuracy Evaluation
- Decision Tree Visualization

## TECHNOLOGIES USED

- Python
- Scikit-Learn
- Pandas
- Matplotlib
- Jupyter Notebook

## DATASET

Iris Dataset

## OUTPUT

- Classification Predictions
- Accuracy Score
- Decision Tree Diagram

## CONCLUSION

The project demonstrates the implementation and visualization of a Decision Tree model using Machine Learning techniques in Python.
